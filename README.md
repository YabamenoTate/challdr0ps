## 🚀 概要

**ﾁｬﾙDr0ps** は、LMS（manaba）上の小テストやドリルの解答データを効率的に収集・共有するためのツールです。
「誰かが解いた解答」をクラウド経由で簡単にシェアします。シェアできるのは「回答が満点になったときのページ」か、提出時に正解が公開される場合は最初の1人が犠牲になり「解答ページ」を表示した場合だけです。よって、ハックされない限りは間違った解答がシェアされることはありません。「みんなの力」に変え、学習（と課題消化）の効率を最大化します。

## ✨ 主な機能

* **📤 シームレスな解答アップロード**
	小テストの解答ページや全問正解ページから、ワンクリックで解答データをサーバーへ送信。
* **📥 インテリジェントな解答ダウンロード**
	過去に誰かが共有したデータがある場合、即座に取得して自分の画面に適用可能。
* **🔄 自動マージ機能**
	PHPバックエンドで複数のユーザからの解答を統合し、より多くの解答キーを構築（50問の中から10問がランダムに出されるようなテストでは、ドロップしたユーザが多いほど効果が出ます）。

## 🛠 テクニカルスタック

### Frontend (Chrome Extension)
* **Manifest V3**: 最新のブラウザ仕様に準拠。
* **Content Scripts**: `manaba` のUIに自然に溶け込むボタンを注入。
* **Background Workers**: `fetch` APIを用いた非同期通信による高速なデータ授受。

### Backend (PHP API)
* **Scalable JSON Storage**: 解答データを `data/` ディレクトリに構造化して保存。
* **Version Control**: 拡張機能のバージョンをチェックし、常に最新のロジックで動作。
* **Security**: `define.php` による簡易プロキシチェックを実装。

## 📖 使い方
1.  **manaba** にログインし、対象のコース・小テストページを開きます。
2.  画面右側に現れる **「アップロード」** または **「ダウンロード」** ボタンをクリック。
3.  データが「ドロップ」され、あなたの課題解決をサポートします。
