## 🚀 概要

**ﾁｬﾙDr0ps** は、LMS（manaba）上の小テストやドリルの解答データを効率的に収集・共有するためのツールです。
「誰かが解いた解答」をクラウド経由で簡単にシェアします。シェアできるのは「回答が満点になったときのページ」か、提出時に正解が公開される場合は最初の1人が犠牲になり「解答ページ」を表示した場合だけです。よって、ハックされない限りは間違った解答がシェアされることはありません。「みんなの力」に変え、強化学習型授業（伝達を学習と誤認する教員による授業）の課題消化の効率を最大化します。

## ✨ 主な機能

* **📤 シームレスな解答アップロード/json形式エクスポート**
	小テストの解答ページや全問正解ページから、ワンクリックで解答データをサーバーへ送信。
* **📥 インテリジェントな解答ダウンロード/json形式インポート**
	過去に誰かが共有したデータがある場合、即座に取得して自分の画面に適用可能。
* **📥 AIに解かせてjsonを出させる用のプロンプトのコピー/AIの解答を貼り付け**
	過去に誰も解いていない小テストを解くとき、最初の犠牲者になる場合は、プロンプトをコピーしてAIに投げるとjsonが出てくるので、それを貼り付けて提出すると9割以上の正答率を目指すことが出来ます。AIの解答を何度も切り貼りしなくとも、jsonをJavaScriptが解析してLMSのテキストボックスに一括入力するので便利です。
* **🔄 自動マージ機能**
	ドロップ用サーバはPHP形式のAPIで構成され、複数のユーザからの解答を統合し、より多くの解答キーを構築（50問の中から10問がランダムに出されるようなテストでは、ドロップしたユーザが多いほど効果が出ます）。

 - 穴埋め問題は穴埋め用テキストボックスの位置を記憶するために前後の文字列の「ハッシュ値」だけを記憶しておくことで、データを軽量化しています。それから、選択問題は選択番号ではなく文字列を保存することで、入れ替わっても対応可能としてあります。
 - 長い文章を書いて回答する問題はコピーされません。

## 🛠 テクニカルスタック

### ブラウザ用Frontend (Chrome Extension)
* **Manifest V3**: 最新のブラウザ仕様に準拠。
* **Content Scripts**: `manaba` のUIに自然に溶け込むボタンを注入。
* **Background Workers**: `fetch` APIを用いた非同期通信による高速なデータ授受。

### サーバ用Backend (PHP API)
* **Scalable JSON Storage**: 解答データを `data/` ディレクトリに構造化して保存。
* **Version Control**: 拡張機能のバージョンをチェックし、常に最新のロジックで動作。

## 📖 使い方
1.  **manaba** にログインし、対象の小テストの回答入力ページ、全問正解の回答完了ページ、解答ページのいずれかを開きます。
2.  回答入力ページの提出の下にあるボタンを押せば、サーバから解答を受け取ったり、jsonから解答をインポートしたり出来ます。全問正解の回答ページや解答ページでは、解答のサーバへのアップロードやjson形式でのエクスポートを行えます。（同じコース内でドロップする場合はサーバ経由が楽ですが、別のコースで使いまわしの小テストがある場合はjson形式が効果的です。）
3.  データが「ドロップ」され、あなたの課題解決をサポートします。
